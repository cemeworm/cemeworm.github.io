分发机制

好的分发机制

人——（优化体验）——>广告<——（提高效率）——信息

好的分发机制指标

  • 召回率（覆盖要多）

  • 准确率（相关性，ctr，cvr）

分发机制——搜索

  • 人=网民搜索词

  • 信息=客户关键词+匹配方式

产品做什么：

1、制定匹配产品规则

  • 覆盖面（召回）

  • 客户接受度

  • 相关性

  • 默认OR主动

2、旧匹配产品规则的策略迭代

  • 搜鲜花有广告，搜xianhua没有广告——迭代中英文

分发机制——信息流

  • 人=不同标签（设备号、兴趣，。。。）

  • 信息=被圈定选中的标签

产品拍卖机制

定义

拍卖也称竞买，商业中的一种买卖方式，卖方把商品卖给出价最高的人

四种典型的拍卖机制

一阶价格密封拍卖

竞拍各方价格保密，同时给出，价高者得

二阶价格密封拍卖

竞标成功者只需要支付第二高的价格

荷兰式拍卖

定高价，逐渐减

英国式拍卖

定低价，逐渐高

互联网广告拍卖机制

广义第一价格GFP（Generalized First Price）

price_i （实收价）= bid_(i)（实出价）

现象：在博弈过程中，各方倾向于试探各广告位的最低价格，价格反复波动

缺点：

  • 平台方收益会非常不稳定

  • 竞价效率不高

广义第二价格GSP

price_i（实收价）= bid_(i+1)（下一位出价）+0.01

现象：后一名选手出紧逼上一名选手的价格，快速消耗前者的广告预算，同时自己的广告成本是由下一名选手决定的

​

CPM的计费方式+GSP计费模式：

price_i = bud_(i+1)+0.01

CPC的计费方式+GSP计费模式：

price_i = ctrq_(i+1)*bid_(i+1)/ctrq_i+0.01

ctrq_iprice_i = ctrq_(i+1)bid_(i+1)+0.01

CPA的计费方式+GSP计费模式：

price_i = cvr-a_(i+1)ctrq_(i+1)bid_(i+1)/ctrq_icvr-a_i+0.01

​

VCG（Vickrey-Clarke-Groves）竞价机制

竞价者对其他竞价者造成的损失来计费的

​

优点：

稳定状态下，按自己的真实预期出价是每位竞价者的强占优策略

缺点：

  • 不好解释

  • 如何正确计算给其他广告主带来的价值损害，并验证

市场保留价

小结

拍卖—定价机制：解决“在有一个位置和给定参与者的出价和预期期望收益后，如何对最后获取这个位置的广告主收取合适的费用”的问题
